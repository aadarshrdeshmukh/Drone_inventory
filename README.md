# 🚁 Drone-Based Inventory Management Dashboard

A premium, technical dossier-style dashboard for analyzing the ROI and financial impact of drone-based inventory management systems.

## ⬢ Project Overview
This application provides a comprehensive analysis of the cost-benefit ratio of implementing drone technology in warehouse environments. It focuses on labor reduction, stock accuracy, and long-term financial projections (NPV/ROI).

## ▥ Key Features
- **Business Analysis**: Tracking labor savings and stock error reduction.
- **Financial Projections**: 5-Year Net Present Value (NPV) and ROI calculations.
- **Interactive Data Explorer**: Real-time filtering of warehouse racks and inventory values.
- **Aesthetic UI**: Custom monochrome "Technical Dossier" design system.

## 🚀 Deployment on Streamlit Cloud

1. **GitHub Upload**: Push this entire directory to a public/private GitHub repository.
2. **Streamlit Cloud**:
   - Log in to [Streamlit Cloud](https://share.streamlit.io/).
   - Click "New App".
   - Select your repository, branch, and main file (`dashboard.py`).
   - Click **Deploy**.

## ▦ Local Setup

```bash
# Clone the repository
git clone <your-repo-link>
cd drone-inventory-dashboard

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run dashboard.py
```

## ▦ Tech Stack
- **Python**: Core logic.
- **Streamlit**: Web framework.
- **Plotly**: Monochrome visual analytics.
- **Pandas**: Financial engine and data manipulation.
